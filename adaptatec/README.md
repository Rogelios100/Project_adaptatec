# 🎓 Adaptatec - Plataforma de Aprendizaje Adaptativo

Una solución web moderna y completa para autoestudio con sistema de recompensas e inteligencia artificial integrada.

## 🚀 Características Principales

### 👥 Tres Tipos de Usuarios
1. **Estudiante (ALU)** - Interfaz para aprendices
2. **Docente (DOC)** - Panel de gestión pedagógica
3. **Administrador (ADMIN)** - Control total del sistema

### 📚 Funcionalidades

#### Para Estudiantes
- ✅ Dashboard con progreso general
- ✅ Gestión de materias (6 materias disponibles)
- ✅ Sistema de recompensas y insignias
- ✅ Seguimiento de puntos y niveles
- ✅ Chat con Asistente IA para resolver dudas
- ✅ Historial de actividad

#### Para Docentes
- ✅ Creación y gestión de cursos
- ✅ Seguimiento de estudiantes
- ✅ Sistema de calificaciones
- ✅ Estadísticas de desempeño
- ✅ Gestión de materias

#### Para Administradores
- ✅ Gestión completa de usuarios
- ✅ Control de cursos del sistema
- ✅ Reportes y estadísticas
- ✅ Auditoría del sistema
