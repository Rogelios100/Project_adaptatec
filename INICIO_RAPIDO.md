# ⚡ INICIO RÁPIDO - Adaptatec con Node.js

## 🎯 En 3 Pasos

### 1️⃣ Instalar dependencias
```bash
npm install
```

### 2️⃣ Iniciar servidor
```bash
npm start
```

### 3️⃣ Acceder a la aplicación
```
http://localhost:3000
```

---

## 📱 Conectar desde otro dispositivo

1. Obtén tu IP local:
   - **Windows (PowerShell):** `ipconfig`
   - **Mac/Linux (Terminal):** `ifconfig`

2. Accede desde otro dispositivo a: `http://<TU-IP>:3000`
   - Ejemplo: `http://192.168.1.100:3000`
