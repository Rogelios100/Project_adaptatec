# 📱 Guía Rápida: Usar Gemini en Adaptatec

## ⚡ En 3 Pasos

### 1️⃣ Configurar tu API Key (Primera Vez)

1. Ve a [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Haz click en **"Create API Key"**
3. Copia tu API Key
4. En Adaptatec, haz click en **"🤖 Configurar IA"**
5. Pega tu API Key
6. Haz click en **"💾 Guardar"**

✅ ¡Listo! Ya tienes Gemini configurado

---

### 2️⃣ Acceder al Chatbot

1. Inicia sesión como alumno (ALU001 / 123)
2. Ve a **"📚 Materias"**
3. Selecciona una materia
4. Haz click en **"💬 Pregunta a la IA"**

---

### 3️⃣ Hacer Preguntas

1. Escribe tu pregunta en el chat
2. Presiona **Enter** o haz click en **"Enviar"**
3. Espera la respuesta de Gemini
4. ¡Disfruta de la explicación! 🎉

---

## 💬 Ejemplos de Preguntas

**"¿Qué es un árbol binario de búsqueda?"**
→ Gemini te da una explicación con ejemplos

**"¿Cuál es la diferencia entre SQL y NoSQL?"**
→ Comparativa completa

**"Ayúdame con este código de JavaScript"**
→ Explica y proporciona mejoras

**"¿Cómo funciona un JOIN en SQL?"**
→ Explicación + ejemplos de queries

---

## 🎯 Consejos para Mejores Respuestas

✅ **Sé específico** 
- ❌ "¿Qué es programación?"
- ✅ "Explícame qué son las clases en programación orientada a objetos"

✅ **Incluye contexto**
- ❌ "¿Cómo hago esto?"
- ✅ "Necesito una función que ordene un array en JavaScript"

✅ **Si no entiendes la respuesta, pregunta más**
- "Explicame eso de forma más simple"
- "Dame un ejemplo de código"
- "¿Cuándo se usa esto en la práctica?"

---

## ⚠️ Notas Importantes

🔒 **Tu API Key es privada**
- Se guarda solo en tu navegador
- No la compartas con nadie
- Adaptatec no la ve

💡 **Límites del Plan Gratuito**
- 60 preguntas por minuto
- Ilimitado por mes
- Perfectamente suficiente para uso educativo

📡 **Necesitas Internet**
- Gemini requiere conexión a internet
- Si se cae, las respuestas vienen del sistema local

---

## ❓ ¿Qué hacer si...?

### "¿Error: API key inválida?"
- Copia bien la API key de Google
- Sin espacios al principio o final
- Intenta probar la conexión primero

### "¿Gemini no responde?"
- Verifica tu conexión a internet
- Intenta otra pregunta
- Recarga la página

### "¿Olvidé mi API Key?"
- Ve a Google AI Studio nuevamente
- Crea una nueva
- Actualiza en la pestaña "🤖 Configurar IA"
